Why AgentEmerald?

⚙️ The only always-on coding agent

Every other tool in this space is a session you open and close. AgentEmerald runs as a persistent background daemon — your sessions survive terminal crashes, your cron jobs fire at 9am, and the agent keeps working while you're away from the keyboard.

📱 Your codebase in your pocket

Through its gateway, AgentEmerald bridges Discord, Telegram, WhatsApp, and Matrix. Ask it to explain a function from your phone on the commute. Have it open a PR while you're away. No other coding agent even attempts this.

🔑 No new subscriptions

AgentEmerald uses GitHub Copilot or GitHub Models — if you've got a GitHub account, you're already authenticated. No OpenAI key, no Anthropic key, no new billing dashboard to manage.

🛠️ Built for terminal veterans

The TUI is built with React/Ink — real streaming markdown, a live tool call sidebar, diff previews before writes, keyboard-driven navigation. It's the coding agent experience that power users have been waiting for.

✅ The verify loop

AgentEmerald auto-detects your project's build, lint, and test scripts with /verify. Add --watch and it reruns the full suite on every file change. No other coding agent has this built in.

📂 Session management that's actually useful

Fork conversations, merge sessions, full-text search across history, export to Markdown or JSON. Your context is yours — not locked inside a subscription service.
