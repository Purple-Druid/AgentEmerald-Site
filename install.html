Install agem

One command to get started. Works on macOS, Linux, and Windows. Node.js and pnpm handled for you.

⚡ Quick Install — macOS & Linux

Run the install script. It detects your OS, downloads the latest release, and adds agem to your PATH.

curl -fsSL https://raw.githubusercontent.com/Purple-Druid/AgentEmerald/main/install.sh | sh

After installation, run:

agem

🪟 Windows — PowerShell

Open PowerShell as Administrator and run:

irm https://raw.githubusercontent.com/Purple-Druid/AgentEmerald/main/install.ps1 | iex

Windows Subsystem for Linux (WSL) is also supported — use the macOS/Linux script inside WSL.

🛠️ Installer Options

Pass flags to the install script to customise behaviour:

Flag Description Default
--dir <path>Install to a custom directory~/.agem
--no-ghSkip GitHub CLI installation checkoff
--no-nodeSkip Node.js installation checkoff
--version <tag>Install a specific release taglatest

Example with flags:

curl -fsSL .../install.sh | sh -s -- --dir /opt/agem --version v0.1.0

🔧 Manual Install

If you prefer full control, clone the repo and build from source:

1. Clone the repository

git clone https://github.com/Purple-Druid/AgentEmerald.git

2. Enter the directory

cd AgentEmerald

3. Install dependencies

pnpm install

4. Build the project

pnpm run build

5. Authenticate with GitHub

gh auth login

6. Start agem

pnpm start

📦 Standalone Binary

Build a self-contained binary that doesn't require Node.js on the target machine:

pnpm run release

Then install it to your PATH:

pnpm run install-bin

The binary is placed in ~/.local/bin/agem (Linux/macOS) or %APPDATA%\agem\agem.exe (Windows).

🚀 First Run

Launch the interactive TUI:

agem

Or start the background gateway daemon first (recommended for persistent sessions and chat bridges):

agem gateway start

Then connect the TUI to the running gateway:

agem

The TUI auto-connects to a running gateway — no extra config needed.

📋 Requirements

The install script handles these automatically. For manual installs, ensure you have:

  • Node.js ≥ 20 — nodejs.org
  • pnpm ≥ 9 — npm install -g pnpm
  • GitHub CLI — cli.github.com (for authentication)

After installing the GitHub CLI, authenticate once:

gh auth login

That's it. No separate API keys, no paid accounts — just a GitHub login.
